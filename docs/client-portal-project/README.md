# REVEL Client Portal Project Hub

Last updated: 2026-05-22
Owner: REVEL Product + Copilot
Status: Active build

## Purpose

This folder is the single source of truth for planning, building, and handing off the REVEL client portal and adjacent vendor/guest experiences.

## Project Outcomes

- Couple outcome: faster planning decisions with fewer surprises.
- Business outcome: higher conversion and package quality with lower coordination overhead.

## Scope Model

- Marketing layer: static website pages for positioning and conversion.
- Portal layer: authenticated, event-scoped role experiences (scaffold created under `portal-app/`).
- Intelligence layer: shared AI + data services for budget, venue fit, and coordination.

## Current Wedge

1. Fusion Flow Experience Architect
2. Venue Production Analyzer

## Access Rule

Only participants in a specific wedding receive access to that wedding's portal modules and data.
