# Project Charter

## Mission

Build the AI operating layer for wedding execution that improves outcomes for couples, vendors, planners, guests, and REVEL internal teams.

## Problem

Planning and execution data is fragmented across calls, chats, docs, and memory. This causes slow decisions, duplicated effort, and day-of risk.

## Goals

- Reduce planning friction and coordination churn.
- Increase confidence in budget and venue decisions.
- Improve event execution quality through shared operational clarity.
- Create a defensible intelligence moat from real event workflows.

## Non-Goals (Phase 1)

- Full CRM replacement
- Generic project management suite
- Wedding marketplace
- Full guest social/engagement app

## Success Criteria

- Couple consult conversion improves.
- Premium package attachment improves.
- Coordination back-and-forth drops measurably.
- Pilot stakeholders report clearer operations with fewer surprises.
